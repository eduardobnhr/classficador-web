DETALHES DO INCIDENTE

INC-2024-8842

Suspicious Login Pattern via API Gateway

Sequência anômala de autenticações detectada no gateway público com padrões compatíveis com phishing e tentativa de enumeração.

DESCRIÇÃO COMPLETA

O sistema correlacionou múltiplas tentativas de login direcionadas a contas administrativas por meio do API Gateway. O padrão inclui variação mínima de payload, uso de user-agents inconsistentes e repetição em janelas curtas de tempo.

A classificação automática indica alta probabilidade de campanha de phishing com tentativa de validação de credenciais. Recomenda-se mitigação imediata e revisão das credenciais afetadas.
